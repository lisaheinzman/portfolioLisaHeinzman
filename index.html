Hive Help

2024

Hive Help is a productivity app where I played a key role in its development, contributing to comprehensive documentation, prototyping, and full-stack development. Built with React for the front end and a robust backend, the app allows users to create personalized profiles and securely store their information in a database powered by Supabase. Key features include an integrated task manager, implementation of React-Calendar for scheduling, and a user-friendly interface designed to enhance productivity. This project showcases a seamless combination of intuitive design and scalable backend functionality.

Pokédex Database

2023

This project is a database system designed to act as a dynamic Pokédex for managing trainers and their Pokémon during battles. The system models a 1-to-n relationship between trainers and their Pokémon, and a similar relationship between trainers and their items. Built using MongoDB for a dynamic database system designed to manage Pokémon and trainers during battles. This project utilizes MongoDB and Express.js to provide a RESTful API for managing trainers, their Pokémon, and their items.

Village

2023

Village is a bartering app that I collaborated on in which we designed to help individuals exchange skills and resources, fostering stronger, more connected communities. As part of a group project, we conducted extensive user research, including task analysis, user testing, and iterative prototyping to identify and address key user needs. By developing detailed user and stakeholder profiles, we ensured the app’s design aligned with the values and goals of our target audience. The project culminated in a high-fidelity prototype, showcasing an intuitive and engaging platform for community-driven collaboration.
